# RAGLens
Retrieval-Augmented Generation Evaluation Platform  

RAGLens visualizes and evaluates retrieval quality across chunking, embedding, and reranking strategies for LLM pipelines.  
Built with **Python**, **LangChain**, and **Next.js + TypeScript**, it aims to make RAG systems more interpretable through structured evaluations and visual analytics.  

## Planned Features
- Modular evaluation backend with LangChain and OpenAI APIs  
- Chunking and reranking comparison dashboard  
- Interactive visualizations of recall, precision, and response faithfulness  
- Batch experiment management and result aggregation  

*(Development in progress – private evaluation code will be released once anonymized)*
